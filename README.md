# AlphaLens Stock Insight

AlphaLens Stock Insight is a stock research platform that helps investors analyze publicly traded companies through technical indicators, peer comparison, historical context, investment scoring, and research-style market insights.

## Features

- Interactive stock charts
- Technical indicators including RSI and moving averages
- Investment scorecard with trend, momentum, valuation, risk, and growth scores
- Separate long-term investor, swing trader, and day trader profile scores
- Market open/closed status and latest data timestamp
- Historical context engine for "hasn't traded this high/low since..." insights
- Peer comparison table and chart
- AI-style executive summary
- Research report generator with markdown download
- Expanded watchlist with major stocks, Canadian banks, and ETFs
- Modular Python architecture using Streamlit components

## Built With

- Python
- Streamlit
- Plotly
- Pandas
- yfinance
